# 🏏 Cricket Game (Bat Ball Stump)

A simple and fun browser-based cricket game built using **HTML, CSS, and JavaScript**.

In this game, the user plays against the computer by choosing:
**Bat 🏏, Ball ⚾, or Stump 🪵**

---

## 🚀 Features

* 🎮 Play against computer
* 🤖 Random computer choice
* 🏆 Win / Lose / Tie result
* 📊 Live score tracking
* 🔄 Reset game option

---

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript

---

## 📂 Project Files

* `index.html` → Game UI structure 
* `style.css` → Styling and layout 
* `script.js` → Game logic and score system 

---

## 🎯 How Game Works

1. Click on **Bat / Ball / Stump**
2. Computer randomly selects its move
3. Result is displayed:

   * ✅ You Win
   * ❌ Computer Wins
   * 🤝 Tie
4. Score updates automatically

---

## ▶️ How to Run Project

1. Download or clone this repository
2. Open `index.html` in your browser
3. Start playing 🎮

---

## 📊 Game Logic (Simple)

* Bat beats Ball
* Ball beats Stump
* Stump beats Bat
