# 🏏 Cricket Game

A simple and interactive Cricket Game built using **HTML, CSS, and JavaScript**.

## 🚀 Features

* 🎮 Play cricket game in browser
* ⚡ Fast and responsive UI
* 🎯 Random score generation
* 📱 Mobile-friendly design

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript

## ▶️ How to Run

1. Download or clone the repository
2. Open `index.html` in your browser
